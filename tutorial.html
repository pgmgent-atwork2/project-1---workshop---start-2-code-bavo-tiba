Welkom in Memory

We gaan een Memory maken!

Daarvoor hebben we drie codetalen nodig

  1. HTML
  2. CSS
  3. JavaScript

Geen paniek, we houden de code eenvoudig, maar maken wel iets superinteractief!

STAP 1 - MAPPEN EN BESTANDEN

  1. Maak een nieuwe map aan (vb op je bureaublad) en geef als naam(Memory)
  2. Open het programma Visual Studio Code. en open de folder die je zopas aangemaakt hebt.
  3. Bouw onderstaande mappen- en bestandenstructuur aan.
  4. Start de live server door rechtsonder de blauwe balk te klikken op Go Live. Jouw standaard browser zal nu gestart worden en je zal een blanco webpagina zien!

STAP 2 - HTML & CSS

Tijd om te coderen, zodat we meer te zien krijgen dan een witte webpagina!

  1. Kopieer onderstaande code in jouw index.html-bestand. Bewaar vervolgens dat bestand.
  2. Kopieer onderstaande code in jouw styles.css-bestand. Bewaar vervolgens dat bestand.

    Dit bevat alvast een aantal basisstijlen die we later zullen nodig hebben voor de look & feel van onze app.

  3. Bekijk het resultaat in de browser. Als alles goed is, ziet het er nu zo uit. Een volledig gekleurd vlak.
